Sjaldersbaum is a game about programming. Unlike most games in the genre, it does not create a new pseudo-language. Instead, players can pick any language of their choosing to solve the problems given.

Dependencies:
* https://github.com/SFML/SFML (2.5.1)
* https://github.com/Tessil/ordered-map (0.8.1)
* https://github.com/jbeder/yaml-cpp (0.6.3)

Precompiled Windows version in SjaldersbaumWin64.zip.
